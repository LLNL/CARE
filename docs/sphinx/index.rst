CARE: CHAI and RAJA Extensions
==============================

Introduction
============
CARE provides extensions to CHAI and RAJA that are designed to make them easier and more portable to use and debug.

Required Dependencies
=====================
* BLT
* CHAI
* RAJA
* Umpire

Optional Dependencies
=====================
* CUDA
* Cub
* LLNL_GlobalID
